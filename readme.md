# Game Tebak Angka untuk Balita Indonesia

## Deskripsi
Game Tebak Angka adalah proyek permainan interaktif berbasis web. Dalam permainan ini, pemain harus menebak angka yang diacak oleh sistem. Proyek ini sedang dalam pengembangan aktif dan baru-baru ini menambahkan fitur sistem penilaian bintang.

## Fitur Utama
- Permainan tebak angka dengan mekanika gameplay inti
- Sistem penilaian bintang untuk mengevaluasi kinerja permainan atau pencapaian pemain
- Antarmuka berbasis web yang interaktif untuk keterlibatan pengguna
- Implementasi menggunakan TypeScript dan JavaScript untuk struktur dan fungsionalitas kode yang lebih baik

## Teknologi yang Digunakan
- **Bahasa Frontend**: 
  - TypeScript (37.2%)
  - JavaScript (35.2%)
  - HTML (7.0%)
  - CSS (20.6%)
- **Konfigurasi**: `tsconfig.json` untuk pengaturan kompilasi TypeScript
- **Pengembangan**: Kompatibel dengan alur kerja paket Node.js
